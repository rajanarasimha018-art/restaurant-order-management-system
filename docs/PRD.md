# Restaurant Order Management System

## Problem Statement

Many small and medium restaurants struggle to manage orders coming from multiple channels.

Orders arrive from:
- Walk-in customers
- QR menu orders
- Swiggy
- Zomato

Restaurant staff often switch between multiple screens and systems, leading to delays, confusion, and inefficient order handling.

---

## Proposed Solution

A unified restaurant order management platform that provides:

- QR-based customer ordering
- Token generation
- Kitchen dashboard
- Order status tracking
- Unified order monitoring

---

## Target Users

- Restaurants
- Cafes
- Bakeries
- Food courts

---

## MVP Features

### Customer

- Scan QR
- Browse menu
- Add items to cart
- Place order
- Receive token number

### Kitchen Staff

- View incoming orders
- Change order status
- Mark order as ready

### Admin

- Manage menu
- Monitor orders
- View daily sales
